# Shaheer Saikhani Portfolio

A personal portfolio website for the internship assignment, built with
Next.js, TypeScript, and Tailwind CSS.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS

## Current Scope

- Multi-page portfolio structure
- Home, About, Skills, Projects, and Contact pages
- Reusable Navbar, Footer, and Hero components
- Typed profile, skills, projects, and contact data
- Responsive foundation for desktop and mobile screens

## Project Structure

```txt
portfolio/
|-- app/
|-- components/
|-- data/
|-- public/
|-- types/
`-- README.md
```

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open the app in your browser:

```txt
http://localhost:3000
```
